# Student Record Management System

A simple Student Record Management System built using Python and Object-Oriented Programming (OOP) concepts. This application allows users to add, search, update, delete, and display student records through a command-line interface.

## Features

* Add new student records
* Search students by Student ID
* Update existing student information
* Delete student records
* Display all student records
* Prevent duplicate Student IDs
* Menu-driven command-line interface

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* Dictionary Data Structure

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-record-management-system.git
```

2. Navigate to the project directory:

```bash
cd student-record-management-system
```

3. Run the program:

```bash
python student_management.py
```

## Available Operations

1. Add Student
2. Search Student
3. Update Student
4. Delete Student
